# Прошивка устройства 

## Общая информация по обновлению програмного обеспечения

Ресурсы для прошивки устройств Meshtastic:

- [Offical Flasher](https://flasher.meshtastic.org/) - Официальная версия веб инстурумента для прошивки и получения прошивок 
- [Russian Custom Flasher](https://mrekin.duckdns.org/flasher/) - Адаптация от русскоязычного сообщества, которые включает как ночные сборки так и релизы с **включенной кириллицей**. А также экспериментальные прошивки с уменьшенным энергопотреблением и улучшенной производительностью. 

    !!! note "Заметка"
        Мы рекомендуем использовать [Russian Custom Flasher](https://mrekin.duckdns.org/flasher/) версию для устройств, которые будут использоваться в Беларуси, так как она оптимизирована для наших условий и поддерживает кириллицу, что может быть полезно для отображения сообщений на русском языке на экране платы.

### Как обновить прошивку?

1. Подключите устройство к компьютеру с помощью USB-кабеля.
2. Откройте выбранный инструмент для прошивки (например, [Offical Flasher](https://flasher.meshtastic.org/) или [Russian Custom Flasher](https://mrekin.duckdns.org/flasher/)).
3. Выберите устройство из списка доступных устройств.
4. Выберите нужную прошивку из списка (рекомендуется использовать последнюю стабильную версию, если вы не тестируете новые функции).
5. Нажмите кнопку "Flash" или аналогичную, чтобы начать процесс прошивки.
6. Дождитесь завершения процесса прошивки. Не отключайте устройство во время прошивки, чтобы избежать повреждения.

### Собрание прошивки самостоятельно (для разработчиков)

Если вы хотите собрать прошивку самостоятельно, вам потребуется установить [PlatformIO](https://platformio.org/) и следовать инструкциям [Developers Documentation](https://meshtastic.org/docs/developers/). Это позволит вам внести свои изменения в код и создать кастомную прошивку для вашего устройства. 
!!! note "Заметка"
    Для большинства пользователей рекомендуется использовать уже собранные прошивки, чтобы избежать возможных проблем с настройкой и сборкой.

!!! danger "Предупреждение"
    Все действия по сборке и использованию собранных Ваши кастомных прошивок выполняются на Ваш страх и риск и находятся под вашей полной ответственностью.
